# Changelog

## [1.0.6] - 2026-02-08

### Added
- **自動化部署**: 新增 GitHub Actions workflow (`daily-crawler.yml`)，支援每日排程與手動觸發。
- **郵件通知**: 實作爬蟲執行結果郵件通知功能。

### Changed
- **全面優化**: 優化所有網站的爬蟲邏輯，提升穩定性與資料準確度。
- **AI 模型**: 更換翻譯與解析模型為 Google Gemini，提升語意理解能力。
- **專案結構**: 清理冗餘測試檔案與 log，優化專案體積。

### Fixed
- 修正部分品牌圖片爬取失敗的問題。
- 修正資料庫重複寫入的判斷邏輯。
